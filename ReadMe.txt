Jogo da Forca - Desenvolvido por Thiago Sérvulo em 21/08/2021
Jogo desenvolvido em linguagem Python usando o QtDesigner para a contrução da interface gráfica.
Uma palavra é sorteada entre as quase 300 palavras que foram introduzidas hardcoded.
O objetivo do jogador é acertar qual palavra é essa antes de ser "enforcado".
O histórico de resultados do jogador são salvos no arquivo txt gerado e consultados no menu de opções.
